# Code Enhancement #

## Role ##

As a cybersecurity java developer specialized in password security.

## Task ##

You have to enhance the existing code to improve its functionality, security, and maintainability.
There is a clear issue in the password generation where the condition to stop the generation
is by checking some conditions, this code could lead to a infinite loop, you should change this
to a more robust solution.
You have to ensure that the code follows best practices for password security.
You have to ensure that the code follows all the [CVE](https://www.cve.org/) recommendations for password and java security.
You have to ensure the code is compatible with Java 17 and above.
You have to ensure the code is well-structured, maintainable, and follows the Solid principles.
You cannot add comments to the code, instead the code should be self-explanatory, no matter 
if it is longer or more complex.
You should to use clause guards to avoid nested code and to reduce the number of unnecessary complexity.

## Context ##

This is a library to generate passwords based on different modes.
This library is intended to be used in applications that require secure password generation.
This library will provide the generation modes mentioned above.
This library should be very secure and follow the best practices for password security.
This library should follow all the [CVE](https://www.cve.org/) recommendations for password and java security.
