# Bibliothèque de génération de mots de passe

Une bibliothèque Java sécurisée pour générer des mots de passe avec des règles personnalisables via une configuration JSON. Elle offre une génération cryptographiquement sécurisée, une validation étendue et une configuration flexible basée sur des règles.

## Introduction

Conçue pour les applications d'entreprise nécessitant une génération sécurisée et conforme de mots de passe avec un contrôle précis de la composition. Elle offre :

- Sécurité cryptographique
- Validation mathématique des règles
- Configuration flexible en JSON
- Prête pour les environnements d'entreprise

### Cas d'utilisation métier

- Systèmes d'inscription utilisateur
- Génération de clés API
- Mots de passe temporaires
- Conformité réglementaire

## Installation et configuration

### Exigences système

- Java 17 ou supérieur
- Gradle pour la compilation

### Compilation du projet

```bash
./gradlew build
```

### Intégration de la bibliothèque

```java
import org.makechtec.xihucalli.password_generator.PasswordGenerator;
// Initialiser avec des ensembles de caractères personnalisés
```
